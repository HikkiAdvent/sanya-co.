# Запуск
> curl -LsSf https://astral.sh/uv/install.sh | sh

Для запуска необходимо установить uv.

[ссылка с инструкцией](https://docs.astral.sh/uv/getting-started/installation/#__tabbed_2_1)

## После утсновки

В папке backend выполнить команды

```brew
uv sync
uv run ./main.py
```

Документация будет по адресу `http://127.0.0.1:8000/docs`

Сам адресс
`http://127.0.0.1:8000`
